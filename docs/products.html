GridDB

"TextPanel storage that transcends traditional data limits."

GridDB transforms humble TextPanels into a sophisticated address-based database system. Using both CustomData and PublicText fields, GridDB creates structured storage that scales across subgrids while maintaining elegant simplicity. Built for engineers who need real data persistence in Space Engineers.

TextPanel-Based Storage

Utilizes both CustomData and PublicText fields of TextPanels for dual-channel data storage with named block addressing.

Address-Based Access

Access data using structured addresses: DB:Domain.Sub.Index.CustomData for precise data location and retrieval.

Subgrid Architecture

Deploy as a dedicated storage subgrid with rotor-mounted TextPanel arrays for scalable database expansion.

Remote Access via IGC

GridDBServer enables cross-grid communication for distributed data access using Inter-Grid Communication.

GridDB in Action

// Initialize GridDB domain
GridDB.Init();

// Add data with automatic organization
GridDB.Set("Domain.Sub.Index.CustomData","some data here",true);
GridDB.Set("Domain.Sub.Index.Text", gameScores,true);

// Retrieve with elegant simplicity
string data = GridDB.Get("Domain.Sub.Index.CustomData");
GridData scores = new GridData("Domain.Sub.Index.Text");
