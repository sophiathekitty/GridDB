## Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sophiathekitty/GridDB.git
   cd GridDB/website
   ```

2. **Add required images:**
   - Review `SCREENSHOTS_NEEDED.md` for complete image requirements
   - Take screenshots in Space Engineers according to specifications
   - Place images in `assets/images/` folder

3. **Optional: Add background music:**
   - Find or create a synthwave/cyberpunk audio file
   - Convert to MP3 format
   - Save as `assets/audio/synthwave.mp3`

4. **Deploy to GitHub Pages:**
   - Push to GitHub repository
   - Enable GitHub Pages in repository settings
   - Set source to main branch / website folder

- **The Presenter**: Chief Visionary Officer, charismatic and delusional
